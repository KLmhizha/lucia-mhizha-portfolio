ABOUT ME

Due to my unlimited passion for learning, one day I found myself on Alison looking for courses to learn. I've always wanted a career that leaves people asking themselves, "what is that?".
My journey started in the beginning of 2022, I took interest in the Data Science course, one day we were learning about R "a programming language that is widely used in data science
and machine learning."

Continuing with my journey, I found out that, there are many programming languages. I chose to try MDN and W3Shcools and thats when I decided to dive into Web Development. This milestone
changed my life completely, the learning was so exhilarating. As time went by, I was now able to implement code and create websites. But I had to keep learning to improve my skills.
Later on a friend recommanded that I try SheCodes "a coding workshop for women", then this year "2023" I continued my learning on SheCodes. It was a great choice, I improved my coding skills and learnt a lot more
about Front-End Development.

The projects that are showcased on my portfolio were built by me, with the use of Git and GitHub to help streamline the development workflow.
